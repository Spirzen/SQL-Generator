# SQL Generator
SQL Generator – инструмент для генерации SQL-запросов с использованием графического интерфейса. 
Проект позволяет пользователям создавать запросы типа SELECT, INSERT, UPDATE, DELETE без необходимости вручную писать SQL-код. 
Это особенно полезно для начинающих разработчиков, аналитиков или тех, кто хочет быстро протестировать свои запросы.

# Основные возможности
поддержка SELECT, INSERT, UPDATE, DELETE;
поддержка JOIN (INNER, LEFT, RIGHT, FULL);
фильтрация данных (WHERE с выбором операторов =, <>, LIKE, IS NULL и др);
возможность скопировать запрос в буфер обмена.

# Как использовать
1. Запустить приложение.
2. Выбрать тип запроса (SELECT, INSERT, UPDATE, DELETE).
3. Указать имя таблицы.
4. Ввести столбцы и значения (если требуется).
5. Настроить фильтрацию, сортировки, соединения и лимит (опционально).
6. Нажать "Сгенерировать запрос".
7. Нажать "Скопировать в буфер обмена".
